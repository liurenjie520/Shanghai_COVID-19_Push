
#### 介绍
这是一个关于上海COVID-19发布会的微信通知系统。

#### 软件架构
Python


#### 安装教程

1. 使用gitaction定时每5分钟运行
2. 输入要推送的微信请求
3. ok

#### 使用说明

1. 注册消息https://wxpusher.zjiecode.com/docs/#/拿到token
2. 替换你的发送链接
3. 点击运行
